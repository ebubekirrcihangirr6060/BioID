# BioID: Biometric Student Attendance System 🆔

BioID is a biometric attendance and tracking system developed using fingerprint technology. This project goes beyond simple coding by focusing heavily on structured software engineering practices, featuring comprehensive requirement analysis and system design.

## 🚀 Overview
Traditional attendance tracking is often inefficient and prone to errors. BioID automates this process by utilizing biometric data (fingerprints) to accurately log student attendance. This repository showcases not only the technical implementation but also the rigorous **Software Engineering** methodologies applied during its development.

## ✨ Key Features
* **Biometric Authentication:** Secure and reliable student identification using fingerprint scanning.
* **Automated Tracking:** Real-time logging of attendance data, eliminating manual entry.
* **Software Requirements Specification (SRS):** A fully detailed SRS document outlining all system constraints, user interactions, and technical requirements.
* **Architecture Design:** Built with scalability and data security in mind.

## 📋 Software Engineering Practices
A major highlight of this project is the emphasis on proper software planning. The included **SRS Document** details:
* **Functional Requirements:** What the system must do (e.g., enroll fingerprint, verify user, log time).
* **Non-Functional Requirements:** Performance, security, and reliability standards.
* **Use Case Diagrams & Workflows:** Visual representations of how students and administrators interact with the system.
